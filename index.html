FitDay - 데일리 코디 추천 서비스

1. 서비스 개요 및 Why

사람들은 매일 아침 옷을 선택할 때 상당한 시간을 소모하며 날씨와 외출 목적에 부합하는 적당한 코디를 결정하기 어려워합니다. "FitDay"는 사용자가 등록한 옷 정보를 바탕으로 기상 상태와 하루 목적에 가장 잘 어울리는 조합을 규칙 기반 논리로 제안함으로써 일상의 소소한 마찰을 덜어내기 위해 기획되었습니다.

2. 핵심 기능 정의 (MVP 범위)

  • 내 옷장 등록 기능: 상의/하의/아우터/신발 등 카테고리별 분류 및 색상, 계절, 두께, 격식 태그 설정
  • 오늘의 조건 수동/자동 입력: 위치 정보 기준 날씨 API 연동 및 현재 기온, 강수 체크
  • 지능형 점수제 추천 알고리즘: 기온별 의류 두께 적합도, 일정별 격식도 가중치 부여
  • 대안 코디 피드백: 추천 결과에 대한 "추웠음", "더웠음", "만족" 피드백에 따른 다음 추천 필터 가속화

3. 구현 및 로컬 데이터 구조

서버리스 환경을 가정하여 사용자의 옷장 데이터, 최근 저장한 코디 및 피드백 로그를 브라우저의 localStorage를 통해 영구 저장 및 유지 관리하도록 설계했습니다.

EcoRoute - 저탄소 대안 경로 서비스

1. 기획 동기

탄소 중립의 중요성이 대두되는 가운데 일상 이동 과정에서의 개별 실천을 유도하는 시스템이 부족합니다. EcoRoute는 최단 시간 위주의 기존 길찾기 솔루션에서 나아가, 탄소 저감 기여를 수치화하고 대중교통과 도보 조합을 적극 장려하는 서비스입니다.

2. 주요 기능 및 아키텍처

  • 실시간 탄소 계산기: 이동 수단(택시, 가솔린 자동차 vs 지하철, 버스, 자전거)별 계수를 활용한 배출 비교
  • 마일리지 보상 체계: 탄소를 가장 많이 아낀 코스로 이동 시 로컬 모빌리티 토큰 포인트 적립 연계
  • 동적 경로 렌더링: Web API 위치 데이터를 반영한 실시간 도보 구간 및 대기 정보 시각화

FocusSpace - 집중력 헬퍼

1. 기획 동기

뽀모도로 타이머 기법은 널리 사용되지만, 단순한 카운트다운은 때로 지루함을 줍니다. 주변 백색소음을 유튜브 등으로 재생할 경우 불필요한 비디오 스트리밍 대역폭이 소모되고 시선이 분산됩니다. FocusSpace는 극소 용량의 브라우저 네이티브 코드로 음원 다운로드 없이 마음을 다잡을 수 있게 돕습니다.

2. Web Audio 기술 구현 세부사항

  • 네이티브 신디사이저: JavaScript의 AudioContext를 선언하여 Pink Noise 및 White Noise 알고리즘을 실시간 버퍼 합성 방식으로 생성
  • LFO 모듈레이터: 빗소리와 바람소리를 재현하기 위해 저주파 필터를 소스 노이즈에 걸어 주기적인 파동 효과 발생
  • 백그라운드 제어: 탭이 비활성화되어도 소리가 끊기지 않고 동작하도록 브라우저 오디오 채널 최적화
메시지가 성공적으로 전달되었습니다!
